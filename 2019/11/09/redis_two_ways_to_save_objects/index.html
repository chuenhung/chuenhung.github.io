Redis保存对象的两种方式

一、序列化(实体需要实现Serializable接口)

  • 序列化就是将一个对象转换为二进制的数据流。这样就可以进行传输，或者保存到文件中。如果一个类的对象要想实现序列化，就必须实现serializable接口。在此接口中没有任何的方法，此接口只是作为一个标识，表示本类的对象具备了序列化的能力而已。
  • 反序列化:将二进制数据流转换成相应的对象
